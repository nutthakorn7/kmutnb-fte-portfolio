## 1. คำอธิบายรายวิชา (Course Description)

ศึกษาเรื่องคัดเฉพาะทางที่ทันสมัยและเป็นที่สนใจในปัจจุบันด้านเทคโนโลยีสารสนเทศและปัญญาประดิษฐ์เพื่อการศึกษา เนื้อหาเปลี่ยนแปลงได้ตามความเหมาะสมในแต่ละภาคการศึกษา สำหรับภาคเรียนนี้เน้น **Generative AI และ Large Language Models (LLMs) เพื่อการศึกษา** ครอบคลุมหลักการทำงาน การประยุกต์ใช้ การ Fine-tune และจริยธรรมของ AI กำเนิดในบริบทการศึกษา

Study of current and emerging selected topics in information technology and artificial intelligence for education. Content varies by semester. This semester focuses on **Generative AI and Large Language Models (LLMs) for Education**, covering working principles, applications, fine-tuning, and ethics of generative AI in educational contexts.

---

## 2. วัตถุประสงค์การเรียนรู้ (Learning Objectives)

เมื่อเรียนจบรายวิชานี้ นักศึกษาสามารถ:

1. **อธิบาย** หลักการทำงานของ Generative AI และ Large Language Models
2. **ใช้งาน** LLMs ผ่าน API และ Prompt Engineering อย่างมีประสิทธิภาพ
3. **Fine-tune** โมเดลภาษาสำหรับงานเฉพาะทางด้านการศึกษา
4. **ออกแบบ** แอปพลิเคชัน AI เพื่อการเรียนการสอน
5. **ประเมิน** คุณภาพและข้อจำกัดของ Generative AI ในการศึกษา
6. **วิเคราะห์** ประเด็นจริยธรรม ความเป็นส่วนตัว และ Academic Integrity ที่เกี่ยวข้อง

## 3. เนื้อหารายสัปดาห์ (Weekly Schedule)

| สัปดาห์ | หัวข้อ | กิจกรรม |
|:---:|---|---|
| **1** | ภาพรวม Generative AI: จาก GPT สู่ Multimodal AI | บรรยาย + Demo: ทดลองใช้ ChatGPT, Claude, Gemini |
| **2** | หลักการทำงานของ LLMs: Transformers, Attention, Tokenization | บรรยาย + Lab: สำรวจ Hugging Face Models |
| **3** | Prompt Engineering สำหรับครูและอาจารย์ | Workshop: ออกแบบ Prompt สำหรับสร้างข้อสอบ, สรุปเนื้อหา, ให้ Feedback |
| **4** | Advanced Prompting: Chain-of-Thought, Few-shot, RAG | Lab: สร้าง RAG System จากเอกสารการสอน |
| **5** | LLM APIs & Application Development | Lab: สร้างแอป EdTech ด้วย Python + OpenAI/Google API |
| **6** | Fine-tuning 101: SFT (Supervised Fine-Tuning) | Lab: Fine-tune โมเดลขนาดเล็กด้วย Google Colab |
| **7** | Fine-tuning ขั้นสูง: LoRA, QLoRA, DPO | Lab: Fine-tune โมเดลสำหรับ Thai Education Task |
| **8** | **สอบกลางภาค + นำเสนอ Midterm Project** | สอบ + Demo |
| **9** | AI for Content Creation: สร้างสื่อการสอน, วิดีโอ, ภาพ ด้วย AI | Lab: ทดลอง DALL-E, Midjourney, Runway, ElevenLabs |
| **10** | AI for Assessment: ตรวจข้อสอบ, ให้ Feedback, Rubric Generation | Lab: สร้างระบบ Auto-Grading ด้วย LLM |
| **11** | AI Tutor & Chatbot Development | Lab: สร้าง Educational Chatbot ด้วย LangChain |
| **12** | Model Evaluation: วัดคุณภาพ LLM สำหรับงานการศึกษา | Lab: Evaluation Metrics — BLEU, ROUGE, Human Eval |
| **13** | AI Ethics in Education: Bias, Hallucination, Academic Integrity | อภิปราย + กรณีศึกษา |
| **14** | นโยบาย AI ในสถาบันการศึกษา: ไทยและนานาชาติ | อภิปราย + ออกแบบ AI Policy สำหรับมหาวิทยาลัย |
| **15** | **นำเสนอ Final Project** | นำเสนอ + Demo ผลงาน |
| **16** | **สอบปลายภาค** | สอบข้อเขียน |
